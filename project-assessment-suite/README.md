# Project Assessment Suite

A comprehensive collection of templates and frameworks for systematically assessing projects in motion, identifying gaps, and implementing corrective actions using leading program management practices.

## Overview

This suite provides program managers with standardized tools to:
- Assess current project state across multiple dimensions
- Define desired future state using best practices
- Conduct systematic gap analysis
- Develop actionable remediation plans
- Track assessment outcomes and improvements

## Assessment Framework

The assessment follows a structured approach:

1. **Current State Assessment** - Evaluate where the project stands today
2. **Future State Definition** - Define where the project should be using leading practices
3. **Gap Analysis** - Identify specific gaps and their impact
4. **Remediation Planning** - Create actionable plans to close gaps
5. **Implementation Tracking** - Monitor progress and measure improvements

## Usage Guidelines

<a id="1-assessment-planning"></a>
### 1. Assessment Planning
- Review the Assessment Planning Guide
- Select appropriate templates based on project type and methodology
- Identify key stakeholders and assessment participants
- Schedule assessment activities

<a id="2-data-collection"></a>
### 2. Data Collection
- Use templates to gather information systematically
- Conduct interviews with key stakeholders
- Review existing project documentation
- Observe current processes and practices

<a id="3-analysis-and-gap-identification"></a>
### 3. Analysis and Gap Identification
- Complete current state assessment using relevant templates
- Define future state using best practice guidelines
- Use the Gap Analysis Matrix to identify and prioritize gaps
- Assess impact and urgency of each gap

<a id="4-remediation-planning"></a>
### 4. Remediation Planning
- Use the Remediation Action Plan template
- Prioritize actions based on impact and feasibility
- Assign ownership and timelines
- Define success metrics

<a id="5-implementation-and-monitoring"></a>
### 5. Implementation and Monitoring
- Track progress using defined metrics
- Regular review and update of action plans
- Communicate progress to stakeholders
- Conduct follow-up assessments

## Best Practices

### Assessment Approach
- **Be Objective** - Focus on facts and evidence, not opinions
- **Be Comprehensive** - Use multiple templates to get complete picture
- **Be Collaborative** - Involve key stakeholders in the assessment
- **Be Action-Oriented** - Focus on actionable improvements

### Data Collection
- **Multiple Sources** - Use interviews, documentation, and observation
- **Triangulation** - Verify findings through multiple data points
- **Documentation** - Keep detailed records of all findings
- **Stakeholder Validation** - Confirm findings with relevant stakeholders

### Gap Analysis
- **Root Cause Focus** - Identify underlying causes, not just symptoms
- **Impact Assessment** - Evaluate business and project impact
- **Feasibility Analysis** - Consider practical constraints and resources
- **Prioritization** - Focus on high-impact, achievable improvements
