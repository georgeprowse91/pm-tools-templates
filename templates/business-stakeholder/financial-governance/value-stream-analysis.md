# Value Stream Analysis Template

### 3. Value Analysis Framework

#### Value Classification
```
Customer Value Activities:
- Directly beneficial to customer
- Customer willing to pay for
- Differentiates from competition
- Improves customer experience
- Solves customer problems

Business Value Activities:
- Required for business operations
- Regulatory/compliance requirements
- Risk mitigation activities
- Infrastructure support
- Quality assurance

Non-Value Activities (Waste):
- Unnecessary movement/transport
- Excess inventory/queues
- Overprocessing/gold-plating
- Waiting/delays
- Defects/rework
- Unused talent/skills
- Overproduction
```

#### Waste Identification (8 Wastes)
```
1. Transportation:
   - Unnecessary movement of materials/information
   - Multiple handoffs
   - Geographic distribution inefficiencies

2. Inventory:
   - Excess work-in-progress
   - Overproduction buffers
   - Information hoarding

3. Motion:
   - Unnecessary movement of people
   - Poor workplace organization
   - Inefficient layouts

4. Waiting:
   - Process bottlenecks
   - Resource unavailability
   - Information delays

5. Overproduction:
   - Making more than needed
   - Working ahead of demand
   - Creating unwanted features

6. Overprocessing:
   - Unnecessary process steps
   - Excessive reviews/approvals
   - Gold-plating solutions

7. Defects:
   - Quality issues requiring rework
   - Information errors
   - Process failures

8. Unused Talent:
   - Underutilized skills
   - Limited empowerment
   - Poor knowledge sharing
```

### 4. Performance Measurement

#### Key Metrics
```
Flow Metrics:
- Lead Time: Time from start to completion
- Cycle Time: Time between completions
- Throughput: Items completed per time period
- Work in Progress (WIP): Items in the system
- Flow Efficiency: Value-add time / Lead time

Quality Metrics:
- First Pass Yield: % completed without rework
- Defect Rate: Defects per unit/transaction
- Right First Time: % correct on first attempt
- Customer Satisfaction: Customer feedback scores
- Internal Quality: Process compliance rates

Cost Metrics:
- Process Cost: Total cost per unit/transaction
- Labor Cost: Direct and indirect labor costs
- Overhead Allocation: Indirect costs per unit
- Cost of Quality: Prevention, appraisal, failure costs
- Cost per Value Hour: Cost per value-adding time
```
