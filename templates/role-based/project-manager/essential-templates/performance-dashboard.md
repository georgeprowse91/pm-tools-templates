# Performance Dashboard Template

## Overview

The Performance Dashboard provides a visual and quantitative representation of key project metrics, enabling stakeholders to quickly understand project health, progress, and performance against objectives.

**Purpose**: Track and visualize key project performance indicators for informed decision-making

**When to Use**: Throughout project execution for regular monitoring and stakeholder reporting

**Who Uses**: Project Manager (owner), Team Members, Stakeholders, Executive Leadership

## Dashboard Customization Guidelines

### Audience-Specific Views

#### Executive Dashboard
**Focus Areas**:
- Overall project health
- Key financial metrics
- Major risks and issues
- Strategic alignment

**Update Frequency**: Weekly
**Detail Level**: High-level summary

#### Project Manager Dashboard
**Focus Areas**:
- Detailed performance metrics
- Team productivity
- Resource utilization
- Operational issues

**Update Frequency**: Daily
**Detail Level**: Detailed operational data

#### Team Dashboard
**Focus Areas**:
- Individual and team performance
- Task completion status
- Blockers and dependencies
- Quality metrics

**Update Frequency**: Real-time/Daily
**Detail Level**: Task and individual level

#### Stakeholder Dashboard
**Focus Areas**:
- Deliverable status
- Schedule adherence
- Quality indicators
- Communication updates

**Update Frequency**: Weekly
**Detail Level**: Outcome-focused

#### Construction/Engineering
**Additional Metrics**:

#### Professional Services
