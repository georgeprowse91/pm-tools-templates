# Program Communications

Templates for program-level communications, reporting, and stakeholder updates.

## What's Included
- Multi-Level Communication Plan (program, project, executive)
- Program Status Reporting Framework
- Stakeholder Communication Calendar
- Escalation and Notification Templates

## Guidance
- Traditional: Formal reporting cadence to governance bodies.
- Agile-at-Scale: Transparent, frequent syncs with demos and inspect-adapt.
- Hybrid: Mix formal status with lightweight, high-frequency updates.
