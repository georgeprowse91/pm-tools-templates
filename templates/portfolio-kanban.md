# Portfolio Kanban

Columns: Intake → Analysis → Prioritization → Approved → In Delivery → Measuring Benefits → Done

Card Fields:
- Title, Owner, Strategic Theme, Expected Benefit, Cost (T-Shirt or $$), Risk, Dependencies, Status, Target Quarter

Policies:
- WIP limits on Analysis and In Delivery
- Definition of Ready/Done per column

Metrics:
- Throughput, Lead Time, % Strategic Alignment, Benefit Realization Rate
