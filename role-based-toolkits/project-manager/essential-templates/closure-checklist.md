# Project Closure Checklist Template

## Overview

The Project Closure Checklist ensures systematic and complete project termination. It provides a comprehensive framework to finalize all project activities, transfer deliverables, release resources, and capture lessons learned.

**Purpose**: Ensure complete and proper closure of all project activities and deliverables

**When to Use**: During project closure phase or when terminating a project early

**Who Uses**: Project Manager, Project Team, Stakeholders, Operations Team
