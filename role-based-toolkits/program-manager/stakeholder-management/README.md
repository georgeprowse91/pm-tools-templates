# Program Stakeholder Management

Templates to identify, analyze, and manage stakeholders and change impact at the program level.

## What's Included
- Stakeholder Analysis and Mapping
- Engagement Strategies by Segment
- Influence/Interest Matrix (Program-Level)
- Change Readiness and Impact Assessment

## Guidance
- Traditional: Formal stakeholder registers and planned communications.
- Agile-at-Scale: Ongoing collaboration with focus on value streams and ART leadership.
- Hybrid: Combine structured mapping with iterative engagement experiments.
