# Portfolio Management Suite

This suite helps Program Managers align initiatives to strategy, prioritize investments, and monitor value realization across the portfolio.

## What's Included

- portfolio-kanban.md — Visual flow of initiatives through intake to delivery
- prioritization-model.md — Weighted scoring model (value, cost, risk, alignment)
- benefits-tracking-dashboard.md — KPIs and benefits realization tracking view
- governance-cadence.md — Portfolio ceremonies and decision forums

## Usage

1. Define strategic themes and scoring criteria.
2. Run intake using the portfolio Kanban and prioritize with the scoring model.
3. Track benefits and risks at portfolio level; escalate via governance cadence.
4. Report progress using standardized dashboards.

## Acceptance Criteria

- Provides at least one template for visualization (Kanban), prioritization, benefits, and governance.
- Cross-links to existing templates where applicable (e.g., risk register, comms).
- Non-breaking addition; no existing paths changed.
