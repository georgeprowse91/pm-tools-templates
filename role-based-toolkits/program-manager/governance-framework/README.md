# Program Governance Framework

Define the decision-making structure, roles, forums, and controls for effective program oversight.

## What's Included
- Governance Charter
- Decision Authority Matrix (RACI/DAF)
- Steering Committee Charter and Cadence
- Change Control Board (CCB) Charter
- Quality Management Approach

## Guidance
- Traditional: Stage-gate reviews, formal approvals, and compliance checkpoints.
- Agile-at-Scale: Lightweight governance aligned to value streams and ARTs; lean approvals.
- Hybrid: Minimal viable governance with risk-based rigor.
