# Program Lifecycle Templates

Phase-aligned guidance and cross-links to canonical templates in this repository. Use these as a program-level map that references existing artifacts under templates/ and project-lifecycle/.

## Phases
- Initiation: link to program_charter_template.md, stakeholder-analysis, business case
- Planning: link to program_management_plan_template.md, roadmap, WBS, resource plan, risk plan
- Execution: change management, integration coordination, quality assurance
- Monitoring & Control: status reporting, risk aggregation, dependency management
- Closure: transition to operations, benefits verification, lessons learned

See also: role-based-toolkits/program-manager/program-planning and portfolio-management.
