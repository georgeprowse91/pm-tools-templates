Use this advisory checklist to ensure consistent governance. Attach this to PRs or link in the PR body.

- [ ] Automated tests passed (unit/integration/regression as applicable)
- [ ] Rollback plan prepared, tested, and documented (link)
- [ ] User acceptance verified (evidence or link)
- [ ] Documentation updated (what changed and where)
- [ ] Change log entry created (summary, timestamp, responsible party)
- [ ] Root Cause Analysis performed (if applicable) and shared
- [ ] Peer review completed (who, when, outcome)
- [ ] Stakeholder notifications sent (audiences, channels, timestamps)
- [ ] Security scan completed; results reviewed
- [ ] Compliance validated (privacy/regulatory/org policies)
- [ ] Accessibility/usability reviewed (if user-facing)
- [ ] Lessons learned recorded; feedback mechanism provided
