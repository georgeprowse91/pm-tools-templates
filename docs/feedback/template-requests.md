# Template Requests

## Request New Templates

Don't see a template you need? Request it here! The community benefits when we create templates that address real project management challenges.

### How to Request a Template

#### Template Request Form

```markdown
## Template Request

**Template Name**: [Proposed name for the template]
**Category**: [Project Phase/Role/Industry/Methodology]
**Priority**: [High/Medium/Low]
**Use Case**: [When would this template be used?]

### Problem Statement
[What challenge does this template solve?]

### Desired Content
[What sections/information should be included?]

### Target Audience
[Who would use this template? Experience level?]

### Success Criteria
[How would we know this template is successful?]

### Similar Templates
[Are there any existing templates that are close but not quite right?]

### Industry/Context
[Any specific industry or project context?]

**Requested by**: [Your Name/Role] (optional)
**Date**: [Request date]
**Urgency**: [Do you need this for an active project?]
```

### Quick Request Format

For simple requests, just tell us:
1. **Template Name**: What should it be called?
2. **Purpose**: What problem does it solve?
3. **Content**: What should be included?
4. **Users**: Who would use it?

## Contributing Your Own Templates

Have a template you've developed that others might find useful?

### Contribution Guidelines
- Follow existing template formats
- Include clear documentation
- Provide usage examples
- Ensure content is original or properly licensed

### What We Look For
- **Practical utility**: Solves real problems
- **Clear structure**: Easy to understand and use
- **Professional quality**: Ready for business use
- **Broad applicability**: Useful to multiple organizations
