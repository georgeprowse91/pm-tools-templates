## Quick Start

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mirichard/pm-tools-templates.git
    ```
2.  **Find your template:**
    Browse the `/templates` directory.

3.  **Customize and go:**
    Open a template in your favorite editor and adapt it to your project.
