# Smartsheet Integration Example

Sync a Status Report template with Smartsheet.

1. Convert `Traditional/Templates/status_report_template.md` to Excel via Pandoc.
2. Upload the Excel file to Smartsheet as a new sheet.
3. Use column types to match project reporting fields.
4. Share the sheet with stakeholders for real-time updates.
5. Export updates back to Markdown for version control if needed.
