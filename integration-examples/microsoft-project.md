# Microsoft Project Integration Example

Import a Work Breakdown Structure into Microsoft Project.

1. Export `Traditional/Process_Groups/Planning/work_breakdown_structure_template.md` to XML using Pandoc.
2. Open Microsoft Project and choose **Open > XML Format**.
3. Map tasks and hierarchy levels during import.
4. Adjust durations and dependencies in Microsoft Project.
5. Save the .mpp file and share with your team.
