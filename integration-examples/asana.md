# Asana Integration Example

Sync an executive dashboard from Excel to Asana tasks.

1. Convert `templates/business-stakeholder/executive-dashboards/Excel/Executive-Dashboard-Workbook.md` to CSV.
2. Import the CSV into Asana using **Add Tasks via CSV**.
3. Assign owners and due dates to each imported task.
4. Use Asana's dashboard view to track completion.
5. Update the Excel workbook from Asana exports for status reporting.
